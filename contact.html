Get in Touch

Have a question, a suggestion, or need help with the app? We're a small operation, but we take every message seriously and will get back to you as soon as we can.

✉️

Email Us

For general questions, feedback, or anything that doesn't fit a bug report — email is the best way to reach us.

Report a Bug

Found something broken? We use GitHub Issues to track bugs and feature requests. Include your device, OS version, and what you were doing.

Open an Issue on GitHub
Not on GitHub? No problem — email us at ZMWApps@outlook.com and we'll handle it from there. Please include your device model, Android/iOS version, and a short description of what happened.

Feature Requests

Have an idea for something that would make the app better? We genuinely want to hear it. Either open a GitHub issue with the label "feature request", or email us directly.

Before submitting, check existing issues to see if someone else has already suggested the same thing — you can add a thumbs up to help us prioritize.
